Rating component

Компонент в виде звездочек, отображающий рейтинг.
У него есть состояние отображения рейтинка, которое содержит текущий рейтинг до изменения. Это нужно для возврата в текущее состояние после hover.
По клику на звезде, происходит изменение рейтинга, для этого используется useEffect
Hover над объектом должен выделять звезды, но после выхода ничего не меняется, если не кликнуть.
 
Компонент имеет два вида отображения 
isEditable - true | false, редактируемый или нет
